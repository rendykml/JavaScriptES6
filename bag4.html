1. Spread pada Array

Spread operator (...) digunakan untuk menggabungkan atau menyalin array.
const combined = [...num1, ...num2];

2. Spread pada Object

Spread operator juga bisa digunakan untuk menggabungkan object.
const combinedObj = {...obj1, ...obj2};
