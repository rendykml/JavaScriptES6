Tutorial Array Method: map, filter, reduce

← Kembali ke Arrow Function
→ Lanjut ke Spread Operator

1. map()

map() digunakan untuk membuat array baru dengan memproses setiap elemen array asal.
const doubleNumber = numbers.map((number) => number * 3);

2. filter()

filter() digunakan untuk membuat array baru berisi elemen yang lolos kondisi tertentu.
const filteredNumbers = doubleNumber.filter((filter) => filter % 3 === 0);

3. reduce()

reduce() digunakan untuk mengakumulasi nilai array menjadi satu nilai saja.
const sum = filteredNumbers.reduce((accumulator, currentValue) => accumulator + currentValue);
