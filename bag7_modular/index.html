Contoh Modular JavaScript

Halaman ini menggunakan type="module" sehingga kamu bisa menggunakan import dan export di file JS.

Penjelasan File app.js

File app.js adalah modul utama yang mengimpor fungsi add, subtract, dan multiply dari util.js menggunakan sintaks import.
Contoh kode di app.js:

import { add, subtract, multiply } from './util.js'; console.log(add(5, 3)); // 8 console.log(subtract(100, 4)); // 96 console.log(multiply(2, 3)); // 6 document.getElementById('output').textContent = 'Modul JavaScript berhasil dijalankan!';

Hasil dari fungsi yang diimpor akan tampil di console browser, sedangkan pesan "Modul JavaScript berhasil dijalankan!" akan tampil di kotak output halaman ini.
