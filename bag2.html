1. Luas Persegi

Arrow function untuk menghitung luas persegi:
const persegi = (sisi) => sisi * sisi;

Contoh penggunaan:
const hasil = persegi(3);

2. Luas Persegi Panjang

Arrow function untuk menghitung luas persegi panjang:
const persegiPanjang = (panjang, lebar) => panjang * lebar;

Contoh penggunaan:
const hasil = persegiPanjang(5, 10);
