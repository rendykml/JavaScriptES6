1. Destructuring Array

Destructuring array memudahkan pengambilan nilai dari array ke variabel.
const [first, second, third, ...rest] = numbers;

2. Destructuring Object

Destructuring object memudahkan pengambilan properti dari object ke variabel.
const { name, age, city: kota, ...otherDetails } = person;
